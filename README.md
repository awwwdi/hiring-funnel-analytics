# 📊 Hiring Funnel Analytics Dashboard

An end-to-end data analytics project to track and optimize hiring funnel performance across multiple sources.

---

## 🚀 Problem Statement

Hiring data was scattered across multiple sheets and weeks, making it difficult to:
- Track conversion rates
- Identify high-performing sources
- Understand funnel drop-offs

---

## 🛠️ Solution

Built a complete analytics pipeline using:

- Python (Pandas) for data cleaning & transformation
- Power BI for interactive dashboarding

---

## 📂 Data Sources

- Master Sheet (lead data)
- Referral sheets
- Multi-week hiring data

---

## ⚙️ Key Features

- Funnel Analysis (Connect → Interview → Pass)
- Source-wise Conversion Tracking
- KPI Metrics:
  - Pass % on Connects
  - Pass % on Interviews
- Weekly Trend Analysis

---

## 📈 Key Insights

- Referral conversion (~85%) is significantly higher than Online (~6%)
- Strong performance at interview stage (~70% pass rate)
- Major drop-off occurs before interview stage

---

## 🧠 Learnings

- Handling messy real-world data (missing values, inconsistent formats)
- Debugging data pipelines and fixing edge cases
- Designing dashboards that communicate insights clearly
